# IIDX Radar Viewer

## 概要

本アプリは音楽ゲーム beatmania IIDX で使用されるレーダ値の一覧表示及び検索が出来るビュワーである。
譜面情報及びレーダ値は[IIDX-Data-Table](https://chinimuruhi.github.io/IIDX-Data-Table/)で公開されている情報を利用する。

### ターゲットユーザ

本アプリで想定されるターゲットユーザは下記とする。

- beatmania IIDXのプレイヤー

## ゲーム各用語

### プレイモード

プレイモードとして下記2つが定義される。プレイモードが変わると同一楽曲で同一難易度でも譜面が異なるため、プレイモード毎にレーダ値が異なる。

| 名称           | 略称 |
| -------------- | ---- |
| シングルプレイ | SP   |
| ダブルプレイ   | DP   |

### 難易度

楽曲別に難易度として下記の5つの難易度が定義されている。各難易度毎に譜面が作成されているが、必ずすべての難易度が存在するわけではなく、BEGINNERやLEGGENDARIAは一部の譜面のみ、ANOTHERがない楽曲等も存在する。

| 名称        | テーマ色 | SPでの略称 | DPでの略称 |
| ----------- | -------- | ---------- | ---------- |
| BEGINNER    | 黄緑色   | SPB        | DPB        |
| NORMAL      | 水色     | SPN        | DPN        |
| HYPER       | 黄色     | SPH        | DPH        |
| ANOTHER     | 赤色     | SPA        | DPA        |
| LEGGENDARIA | 紫色     | SPL        | DPL        |

### レーダ値

レーダ値とは下記の6つの譜面構成要素を数値化した値で0.00～200.00の小数第2位までを持つ実数値である。

| 名称    | テーマ色   | レーダチャート上の位置 |
| ------- | ---------- | ---------------------- |
| NOTES   | ピンク色   | 0時                    |
| PEAK    | オレンジ色 | 2時                    |
| SCRATCH | 赤色       | 4時                    |
| SOF-LAN | 水色       | 6時                    |
| CHARGE  | 紫色       | 8時                    |
| CHORD   | 黄緑色     | 10時                   |

### ノーツレーダ

譜面のレーダ値をレーダチャートで表現したグラフをノーツレーダと呼ぶ。beatmania IIDXプレイヤはゲーム内で表示されるレーダチャートに対して主に下記を目的としてゲームプレイに活用する。

- 譜面傾向の把握
- プレイヤノーツレーダ (プレイヤ自身の実力指標) の更新

## 機能要件

- [ ] 譜面別レーダ値一覧表示機能: 譜面（プレイモード×楽曲×難易度）別に各レーダ値を表示
- [ ] フィルタ機能: 楽曲名、レベル、BPM、各レーダ値等の属性に対するフィルタ
- [ ] ソート機能: 楽曲名、レベル、各レーダ値に対するソート
- [ ] 検索条件・ソート条件の保存・復元機能: ブラウザのLocalStorageに保存し、次回アクセス時に復元
- [ ] 検索条件リセット機能: すべての検索条件を初期状態に戻す

## 画面構成

画面構成は下記とする。

- 譜面別レーダ値一覧表示画面：フィルタ及びソートが可能な譜面別にレーダ値を一覧表示する画面
